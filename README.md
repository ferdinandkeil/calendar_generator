# Calendar Generator Plugin for Inkscape

This plugin was originally written by [George Zhang](https://georgexyz.com/python-calendar-app.html). Thank's a bunch for this :+1: !

You can find the original code on George's website ([Python Code To Generate Full Size Yearly Calendar in Inkscape](https://georgexyz.com/python-calendar-app.html)).

I made some changes so the extension would run in Inkscape 1.1. I also added some additional options like a language selection.
